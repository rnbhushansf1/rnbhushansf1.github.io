Telco Customer Churn Prediction

End-to-End MLOps · Azure ML · FastAPI · Streamlit

This project builds an intelligent system that predicts which customers are likely to stop using a service — giving businesses an early warning so they can act before losing them. It automatically learns from historical customer data, compares multiple prediction models, and always keeps the best-performing one live. The system updates itself every day with fresh data, so predictions stay accurate over time without anyone having to manually retrain or redeploy anything. A live dashboard lets anyone monitor how the system is performing, spot unusual patterns, and see each customer's churn risk at a glance.
